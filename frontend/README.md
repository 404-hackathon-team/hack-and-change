# Фронтенд приложение для хакатона - EduApp

Это веб приложение разработано с использованием Nuxt 4.

## Структура проекта:
- `app/`>
- `components/` - Vue компоненты, используемые в приложении.
- `pages/` - Страницы приложения, которые автоматически маршрутизируются.
- `assets/` - Статические файлы.
- `composables/` - Повторно используемые функции и логика.
- `layouts/` - Макеты страниц.
- `app.vue` - Главный компонент приложения.

## Используемые технологии:
- [Nuxt 4](https://nuxt.com/) - Фреймворк
- [Vue 3](https://vuejs.org/) - Фреймворк
- [TypeScript](https://www.typescriptlang.org/) - Язык программирования
- [UnoCSS](https://unocss.dev/) - Утилитарный CSS фреймворк

## Установка и запуск проекта

1. `bun install` - Установить зависимости
2. `bun run dev` - Запустить приложение в режиме разработки
3. Открыть [http://localhost:3000](http://localhost:3000) в браузере

## Сборка проекта
- `bun run build` - Собрать проект для продакшена

- `docker compose up --build` - Собрать и запустить проект в Docker контейнере (порт 3000)
